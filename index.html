Quelques commandes de Git

-Suprimer un fichier grâce à la balise "git rm" pour que la suppresion soit indexé

    Quelques spécifications
  • git rm -f force la suppresion d'un fichier modifié et indexé
  • git rm log/\*.log supprime tous le fichiers avec l'extension ".log" se situant dans le dossier "log/"
  • git rm \*~ suprrime tous fichiers se terminant par "~"

-Pour sauvegarde(crée un nouveaux point "d'engrage") on utillise git commit

  • git commit -m "texte" pour ecrire directement le message (en Anglais)

-Pour commencer le projet (créer la base) on utillise "git inti" [on peut/doit aller directement dans le(s) (sous-)répertoire ou on veut le créer]

-Pour sauvegarder(commencer à suivre) un dossier on utilise la commande git add "plus nom du fichier"

  • on peut aussi fait git add . on peut sauvegarder le fichier courant (tout ce qui a à l'interieur)

-Pour crée une nouvelle branche on utitlise git branch "nom de la branche"

  • pour changer de branche on fait git checkout "nom de la branche" ne pas oublier de faire un git commit avant de changer de branche
  • pour "fusionner" (relier les informations) git merge "nom de l'autre fichiers" (d'abord se placer sur la branche qui reçoit la fusion!)

Pour ramener une information a distance

  1. tout d'abord crée un lien avec "git remode add leNomQuOnVeut url"
  2. Pour tirer les informations "git pull leNomDuliens LaBrancheEnQuestion"

Pour envoyer une informationà distance

  1. tout d'abord faire un "git commit -m"
  2. Pour envoyer l'information git push LeNomDuLiens LaBrancheEnQuestion

Quelques commandes de Git

-Suprimer un fichier grâce à la balise "git rm" pour que la suppresion soit indexé

    Quelques spécifications
  • git rm -f force la suppresion d'un fichier modifié et indexé
  • git rm log/\*.log supprime tous le fichiers avec l'extension ".log" se situant dans le dossier "log/"
  • git rm \*~ suprrime tous fichiers se terminant par "~"
